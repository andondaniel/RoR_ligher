Spylight RESTful API - See individual resource pages for information on :slug formatting guidelines

Resources

Characters
Characters arranged by the show in which they appear

Resource Description
GET /api/v1/shows/:show_slug/characters Lists all characters for a given show
GET /api/v1/shows/:show_slug/characters/:slug Fetches details of a single character
POST /api/v1/shows/:show_slug/characters Creates a character with a set of optional attributes
PATCH /api/v1/shows/:show_slug/characters/:slug Update an existing character
DELETE /api/v1/shows/:show_slug/characters/:slug Delete an existing character

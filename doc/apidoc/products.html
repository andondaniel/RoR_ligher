Products
Products seen on our supported shows.

The slugs used to identify products are the id of the product followed by the downcased name of the product where all spaces have been replaced with hyphens

Supported Formats

json
>>>

GET /api/v1/products
List all products

GET /api/v1/shows/:show_slug/products
Lists all products for a given show

GET /api/v1/shows/:show_slug/characters/:character_slug/products
Lists all products for a given character

GET /api/v1/shows/:show_slug/episodes/:episode_slug/products
Lists all products for a given episode

GET /api/v1/shows/:show_slug/episodes/:episode_slug/scenes/:scene_id/products
Lists all products for a given scene

Array of all relevant products. Individual nodes within the array map onto JSON representations of products

Supported Formats

json

GET /api/v1/products/:product_slug
Display the details of a given product

GET /api/v1/shows/:show_slug/products/:product_slug
Display the details of a given product

GET /api/v1/shows/:show_slug/characters/:character_slug/products/:product_slug
Display the details of a given product

GET /api/v1/shows/:show_slug/episodes/:episode_slug/products/:product_slug
Display the details of a given product

GET /api/v1/shows/:show_slug/episodes/:episode_slug/scenes/:scene_id/products/:product_slug
Display the details of a given product

Details included are: id, name, description, approved(whether or not an admin as approved the item to go live), slug, a hash of colors the product is available in(keys correspond to color codes), an array of product_images, and an array of product_sources

json

POST /api/v1/products
Create a new product

POST /api/v1/shows/:show_slug/products
Create a new product

POST /api/v1/shows/:show_slug/characters/:character_slug/products
Create a new product

POST /api/v1/shows/:show_slug/episodes/:episode_slug/products
Create a new product

POST /api/v1/shows/:show_slug/episodes/:episode_slug/scenes/:scene_id/products
Create a new product

Used for creating a new product within the given scope.

Supported Formats

json

Params

Param name Description
product
optional , nil allowed
Value: Must be a Hash
product[name]
required

Humanized name of the product


Value: Must be String
product[description]
optional , nil allowed

Textual description of the item


Value: Must be String
product[approved]
optional , nil allowed

Boolean marker of whether or not the item has been approved by an admin


Value: Must be one of: true, false.

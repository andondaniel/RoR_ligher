Scenes
Scenes grouped by episode

Supported Formats

json
>>>

GET /api/v1/shows/:show_slug/episodes/:episode_slug/scenes
Lists all scenes for a given episode

Displays all the scenes for a given episode. Scenes will contain an array of products that appear within them.

Supported Formats

json

Examples

GET /api/v1/shows/new-girl/episodes/new-girl-season2-episode7/scenes
200
{
  "scenes": []
}
>>>

GET /api/v1/shows/:show_slug/episodes/:episode_slug/scenes/:scene_id
Fetches a particular scene

Displays the start and end time of a given scene as well as the products that appear during that time span.

Supported Formats

json
>>>

POST /api/v1/shows/:show_slug/episodes/:episode_slug/scenes
Creates a new scene within a given episode

Used to create a new scene within a given episode. Returns the JSON representation of the altered episode

Supported Formats

json

Params

Param name Description
scene
optional , nil allowed
Value: Must be a Hash
scene[start_time]
optional , nil allowed

Start time of the scene relative to the beginning of the episode. Measured in seconds.


Value: Must be Fixnum.
scene[end_time]
optional , nil allowed

End time of the scene relative to the beginning of the episode. Measured in seconds.


Value: Must be Fixnum.
>>>

PATCH /api/v1/shows/:show_slug/episodes/:episode_slug/scenes/:scene_id

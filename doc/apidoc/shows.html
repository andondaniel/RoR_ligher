Shows
Supported Television Shows

The slugs used to identify shows are simply the downcased name of the show where all spaces have been replaced with hyphens

Supported Formats

json
>>>

GET /api/v1/shows
Lists all shows

Returns an Array of all supported shows. Each show in the array contains and id, name, and slug, as well as nested objects relevant to that show. E.g., each show lists its characters and episodes.

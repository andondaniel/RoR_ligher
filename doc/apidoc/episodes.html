Episodes
Episodes grouped by show

The slugs used to identify episodes are of the following format:

The slug of the associated show, hyphen, the word season, value of the season variable, hyphen, the word episode,  value of the episode_number variable. e.g.,
new-girl-season1-episode1

Supported Formats

json
>>>

GET /api/v1/shows/:show_slug/episodes
Lists all episodes for a given show

Returns an Array of all episodes of the selected show. Each episode in the array contains nested objects relevant to them. E.g., each episode will contain an array of scenes and an array of episode links (links to video streams of the episode).

Supported Formats

json

POST /api/v1/shows/:show_slug/episodes
Create a new episode

Create an episode for a given show. Will return a JSON representation of the show for which you just created an episode

Supported Formats

json

Params

Param name Description
episode
optional , nil allowed
Value: Must be a Hash

GET /api/v1/shows/:show_slug/episodes
Lists all episodes for a given show

Returns an Array of all episodes of the selected show. Each episode in the array contains nested objects relevant to them. E.g., each episode will contain an array of scenes and an array of episode links (links to video streams of the episode).

Examples

GET /api/v1/shows/new-girl/episodes
200

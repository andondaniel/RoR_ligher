GET /api/v1/products
List all products

GET /api/v1/shows/:show_slug/products
Lists all products for a given show

GET /api/v1/shows/:show_slug/characters/:character_slug/products
Lists all products for a given character

GET /api/v1/shows/:show_slug/episodes/:episode_slug/products
Lists all products for a given episode

GET /api/v1/shows/:show_slug/episodes/:episode_slug/scenes/:scene_id/products
Lists all products for a given scene

Array of all relevant products. Individual nodes within the array map onto JSON representations of products

Examples

GET /api/v1/shows/new-girl/characters/jess/products

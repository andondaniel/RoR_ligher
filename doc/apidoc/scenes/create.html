POST /api/v1/shows/:show_slug/episodes/:episode_slug/scenes
Creates a new scene within a given episode

Used to create a new scene within a given episode. Returns the JSON representation of the altered episode

Supported Formats

json

Params

Param name Description
scene
optional , nil allowed
Value: Must be a Hash
scene[start_time]
optional , nil allowed

Start time of the scene relative to the beginning of the episode. Measured in seconds.


Value: Must be Fixnum.
scene[end_time]
optional , nil allowed

End time of the scene relative to the beginning of the episode. Measured in seconds.


Value: Must be Fixnum.
